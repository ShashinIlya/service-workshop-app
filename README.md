# Service Workshop App

Десктопное приложение для учёта работы сервисной мастерской.

Проект реализует базовую информационную систему сервисного центра:
работу с клиентами, мастерами и заказами на диагностику и ремонт.

---

## Основной функционал
- Авторизация пользователей
- Управление клиентами (добавление, редактирование, удаление)
- Управление мастерами
- Учёт заказов и диагностик
- Работа с базой данных SQLite
- Реализация CRUD-операций
- Графический интерфейс на Tkinter

---

## Технологии
- Python 3
- Tkinter / ttkbootstrap
- SQLite
- SQL

---

## Архитектура и структура проекта
Проект построен с разделением логики:
- пользовательский интерфейс
- работа с базой данных
- бизнес-логика приложения

База данных нормализована и содержит таблицы:
- users
- clients
- masters
- orders
