# Service Workshop App 

Десктопное приложение для учёта работы сервисной мастерской. Автоматизирует управление клиентами, заказами и персоналом.


# Основной функционал
 Авторизация: Разграничение прав доступа.
 CRM-модуль: Полное управление базой клиентов (CRUD).
 Управление персоналом: Учёт мастеров сервиса.
 Диспетчеризация: Создание и контроль заказов на диагностику и ремонт.
 GUI: Современный графический интерфейс на базе `Tkinter` с использованием тем `ttkbootstrap`.


# Структура базы данных
Используется реляционная модель данных с поддержкой связей «один ко многим»:
Clients ↔ Orders: один клиент может иметь историю из нескольких заказов.
Masters ↔ Orders: за каждым мастером может быть закреплено несколько активных работ.


# Технологии
 Python 3.x
 Tkinter / ttkbootstrap
 SQLite 3
 Прямые SQL-запросы для управления данными
